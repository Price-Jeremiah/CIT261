Variables:

Variables are simply containers for storing data:

var x = 10;
var y = 11;

In this example, the "var" indicates to the code that the letter next to it will be an assigned value.

"variable" "x" is now "10"
"variable" "y" is now "11"

We can now use these variables to set parameters and make loops, statements, and functions. Another example:
